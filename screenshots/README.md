# Screenshot Placeholders

This directory contains screenshots for the README.

## Required Screenshots

- `terminal-demo.png` - Main terminal demo screenshot
- `mock-mode.png` - Screenshot of mock mode in action
- `real-terminal.png` - Screenshot with real terminal (godotty-node)

## How to Add Screenshots

1. Run the project in Godot Editor
2. Press F12 or use OS screenshot tool
3. Save as PNG to this directory
4. Update README if needed
